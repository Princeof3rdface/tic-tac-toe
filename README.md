# Tic Tac Toe Game 🎮

A simple Tic Tac Toe game built using Python. This project allows two players to play against each other in the terminal.

## 🚀 Features
- Two player mode (X and O)
- Input validation
- Win/draw detection
- Simple CLI interface

## 🛠️ Technologies Used
- Python 3
